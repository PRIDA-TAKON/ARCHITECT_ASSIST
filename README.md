# 🏛️ ARCHITECT_ASSIST: Your AI Design Partner

**ARCHITECT_ASSIST** คือผู้ช่วยสถาปนิกอัจฉริยะ (AI Agent) ที่รันบนเครื่องของคุณเอง (Local) เพื่อช่วยจัดการงานถึกๆ (Tedious Tasks) ในการอ่านแบบร่าง CAD (.dxf) และการทำตารางประมาณราคาก่อสร้าง (BOQ) ใน Excel (.xlsx)

## ✨ ฟีเจอร์เด่น
*   **CAD Analysis:** สั่งให้ AI อ่านข้อมูลปริมาณงาน (Quantity Take-off) จากไฟล์ `.dxf` เช่น ความยาวผนัง, จำนวนเสา, หรือพื้นที่ห้อง ได้ทันที
*   **BOQ Automation:** สั่งให้ AI กรอกข้อมูลราคา, ปริมาณวัสดุ หรือแก้ไขรายการในไฟล์ Excel (Bill of Quantities) ผ่านการแชท
*   **Modern LLM Support:** รองรับสมองกลรุ่นล่าสุดจาก Google (Gemini 1.5 Pro/Flash)
*   **Self-Healing Runner:** ระบบรันโปรแกรมอัจฉริยะ ตรวจสอบความพร้อมของเครื่องและติดตั้งสิ่งที่ขาดให้อัตโนมัติ (ไม่ต้องมีความรู้เรื่องโค้ด)
*   **Professional Identity:** มาพร้อมไอคอน **"AA" สีฟ้าบนพื้นหลังดำ** สังเกตง่ายบนหน้าจอ Desktop

## 🚀 วิธีติดตั้ง (Step-by-Step Guide)

เพื่อให้โปรแกรมทำงานได้อย่างสมบูรณ์ กรุณาทำตามขั้นตอนดังนี้:

1.  **ดาวน์โหลดโปรแกรม:** กดปุ่ม **"Code"** > **"Download ZIP"** จากหน้า GitHub นี้
2.  **สำคัญ! ต้องแตกไฟล์ก่อน:** 
    *   คลิกขวาที่ไฟล์ `.zip` ที่โหลดมา 
    *   เลือก **"Extract All..." (ถอนการติดตั้งทั้งหมด)** 
    *   *ห้ามรันตัวติดตั้งจากในหน้าต่าง Zip โดยตรง*
3.  **เริ่มการติดตั้ง:** 
    *   เข้าไปในโฟลเดอร์ที่แตกไฟล์ออกมาแล้ว 
    *   ดับเบิลคลิกไฟล์ **`INSTALL_WINDOWS.bat`**
    *   รอจนระบบติดตั้งส่วนประกอบ (Python & Libraries) จนเสร็จสิ้น
4.  **เข้าใช้งาน:** 
    *   ไปที่หน้าจอ Desktop ของคุณ 
    *   ดับเบิลคลิกไอคอน **"AA" สีฟ้า** เพื่อเริ่มใช้งานได้ทันที!

## 🛡️ ระบบตรวจสอบข้อผิดพลาด (Self-Healing)
โปรแกรมเวอร์ชันใหม่มีระบบป้องกันปัญหาที่พบบ่อย:
*   **Zip Detection:** ระบบจะแจ้งเตือนและไม่ยอมให้ติดตั้งหากคุณยังไม่ได้แตกไฟล์ เพื่อป้องกันไอคอน Shortcut พัง
*   **Path Correction:** รองรับการติดตั้งในโฟลเดอร์ที่มีเว้นวรรคและชื่อภาษาไทย
*   **Auto-Repair:** หากไฟล์ระบบหรือ Library หายไป ระบบจะพยายามซ่อมแซมตัวเองให้โดยอัตโนมัติ
